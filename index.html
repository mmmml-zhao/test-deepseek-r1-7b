MCP协议详情

MCP(大模型上下文协议)是一种用于管理和维护与大模型交互的上下文状态的标准协议。

请求示例:

{
  "session_id": "sess_123456",
  "message": "解释量子计算",
  "context": {
    "language": "zh-CN",
    "tone": "professional"
  }
}

响应示例 (流式):

data: {"content": "量子计算", ...}
data: {"content": "是一种利用", ...}
data: {"content": "量子力学原理的计算方式", ...}
